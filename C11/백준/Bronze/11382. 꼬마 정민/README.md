### 문제 설명

<p>꼬마 정민이는 이제 A + B 정도는 쉽게 계산할 수 있다. 이제 A + B + C를 계산할 차례이다!</p>

### 입력 

 <p>첫 번째 줄에 A, B, C (1 ≤ A, B, C ≤ 10<sup>12</sup>)이 공백을 사이에 두고 주어진다.</p>

### 출력 

 <p>A+B+C의 값을 출력한다.</p>

## 현성의 문제 풀이
입력란에 보면 A, B, C (1 ≤ A, B, C ≤ 10<sup>12</sup>) 이런 조건이 있다.
int 정수형은 4byte로 약 -21억 부터 21억 까지의 범위가 있다
long
long long 정수형은 8byte로 int형 보다 더 많은 약 -9경 부터 9경까지의 범위를 가지고 있다.
따라서 이 문제는 long long형을 쓰는게 더 올바른 선택지이다.
